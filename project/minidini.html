MiniDini ist eine Webanwendung, die es den Benutzern ermöglicht, mühelos über das Inventar in gemeinsam genutzten Haushalten, sei es eine Wohngemeinschaft oder eine Familie, Überblick zu behalten.

MiniDini is a web application that allows users for effortless inventory tracking for shared households, be it a flatshare or family home.

MiniDini ist eine fiktive Anwendung für den Detailhändler Migros Genossenschaft Schweiz. Die Migros ist in keiner Weise in das Projekt involviert.

Durch die Anwendung können die Benutzer den Überblick über gemeinsame Investitionen behalten und sicherstellen, dass jeder fair behandelt wird, falls jemand einen Haushalt verlässt. Gegenstände können dem Inventar eines Haushalts hinzugefügt und der Eigentumsanteil unter den Bewohnern eingestellt werden. Wertminderungen im Laufe der Zeit werden automatisch berechnet. Innovative Funktionen ermöglichen eine einfache Aufteilung des Eigentums beim Erwerb eines neuen Gegenstands oder beim Verlassen des Haushalts.

MiniDini is a fictional application for the retailer Migros Genossenschaft Schweiz. Migros is in no way involved in the project.

Through the application users can keep track of shared investements and make sure everyone gets treated fairly in case of someone leaving a household. Items can be added to a households inventory and property divided among inhabitants. Value impairment over time calculated automatically. Innovative features allow easy property division when acquiring a new item or leaving.
